# Change Log

All notable changes to the "builder-lang" extension will be documented in this file.

## [2.0.1] - 2025-11-03

### Fixed
- `.builderignore` file icon now displays automatically without requiring icon theme activation
- Icons now work with any existing icon theme (no longer replaces your preferred icon theme)

### Removed
- Standalone icon theme contribution (no longer needed)

## [2.0.0] - 2025-10-XX

### Added
- Full Language Server Protocol support
- Autocomplete for fields, types, and dependencies
- Real-time diagnostics and validation
- Go to definition, hover, and find references
- Rename refactoring across all Builderfiles

## [1.0.0] - 2025-10-26

### Added
- Initial release
- Syntax highlighting for Builderfile and Builderspace files
- Custom file icon for Builder configuration files
- Language configuration with auto-closing pairs
- Comment support (line and block)
- Bracket matching and auto-indentation
- Code folding support
- Support for Builder DSL keywords, properties, and types
